# クイックスタート: デモページ一覧とTodo CRUD

## 前提

- Flutter SDKがセットアップ済み
- 依存関係取得済み

## 起動

```bash
fvm flutter pub get
fvm flutter run
```

## 動作確認

1. ホーム画面を開き、デモページ一覧が表示されることを確認する
2. 一覧から「TODO CRUD」ページへ移動する
3. TODOを追加し、一覧へ反映されることを確認する
4. TODOを編集し、一覧に更新内容が表示されることを確認する
5. TODOを削除し、一覧から消えることを確認する
6. アプリを再起動し、TODOが保持されていることを確認する
7. 一覧から「StatefulShellRoute」ページへ移動する
8. セクションを切り替えて戻り、状態が保持されることを確認する
